Welcome to my website!

This website will house all the rules required to play the tabletop roleplaying game I am creating! Right now none of the pages are actually finished and will change often as I playtest the game.


The purpose of this game is to give players the freedom to create whatever type of character they want, and be able to play it effectively! There are no classes so characters are defined by their race, background traits, feats, themes, combat styles, combat and spellcasting abilities, and of course, their personality! There are virtually unlimited ways to customize your character! Because there is so much information to filter through, I will create a search database which you can use to quickly find the options you are looking for: -database has yet to be made-

The second goal of this game is to be more realistic and intuitive than other roleplaying games. Combat flows more smoothly, skills are more evenly and comprehensively laid out, and the weapons/armor system closely resembles real-world combat without being overly complicated.

If you are looking for a highly customizable game system with easy to learn rules, realistic mechanics, and an emphasis on flexible gameplay, this is the game for you!
